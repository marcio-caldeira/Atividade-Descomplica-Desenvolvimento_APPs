Atividade Prática: Criação e Gerenciamento de um Repositório no GitHub


Atividade Prática:

        A atividade prática da Aula 10, da Disciplina: Prática Integradora Desenvolvimento de Apps, propõe a Criação e Gerenciamento de um Repositório no GitHub. A atividade em questão visa ao desenvolvimento de habilidades no uso do GitHub para criar repositórios e gerenciar versões de projetos. Para a execução desta atividade foi preciso de pesquisa online, conta no GitHub e conhecimentos básicos de comandos Git. A atividade em si envolve a prática do Git, um sistema de controle de versão distribuído, e do GitHub, uma plataforma de hospedagem colaborativa. O objetivo é capacitar os participantes a trabalhar em projetos compartilhados de maneira eficiente e integrada.

  1. Criação de um Repositório GitHub;
  2. Configuração do Projeto Local;
  3. Conexão do Repositório Local com o GitHub;
  4. Adição e Commit de Alterações;
  5. Push para o Repositório Remoto;
  6. Adição de Imagens ao Projeto.
